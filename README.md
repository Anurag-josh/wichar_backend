# wichar_backend

Backend server for the Wichar Medicine Reminder application.

## Features
- User creation and linking (Patient/Caregiver)
- Medicine management (Add, Edit, Delete, Mark Taken/Missed)
- Cloudinary integration for medicine images
- Notification system

## Setup
1. Clone the repository
2. Run `npm install`
3. Create a `.env` file with MongoDB and Cloudinary credentials
4. Run `npm start` or `node server.js`
